17.1.1 Probability

You roll a fair, six-sided, die repeatedly until one of the following events occurs:

  1. The roll is a one
  2. You have rolled the die four times

Your random variable \(X\) is the number of time you roll the die.

  1. What is the support of \(X\)? Is \(X\) a discrete or continuous random variable?

  2. What is the probability that \(X=2\)? Note that this is the probability that the first roll is not a one, and the second roll is a one.

  3. What is the probability mass function or probability density function of \(X\)?

  4. Calculate \(E[X]\)

  5. Calculate \(V[X]\)

17.1.2 Inference

You have a sample \(\{X_i\}_{i=1}^N\) and calculate the sample mean \(\bar x=\frac{1}{N}\sum_{i=1}^NX_i=2.7\). Youtest the following null hypothesis against the alternative hypothesis:

\[ H_0:\ E[X]=3,\quad H_A:\ E[X]\neq 3 \]

using the following test statistic:

\[ z=\left(\frac{\bar x}{3}-1\right) \]

The cumulative density function when \(H_0\) is true is: \[F(x)=(1+\exp(-x))^{-1}\]

and so the inverse cumulative density function (solving \(F(x)=p\)) is:

\[ F^{-1}(p)=\log(p)-\log(1-p) \]

  1. Choose the correct statement, and explain why it is true:

    1. We will reject the null hypothesis when \(z\) is sufficiently positive
    2. We will reject the null hypothesis when \(z\) is sufficiently negative
    3. We will reject the null hypothesis when \(|z|\) is sufficiently large
    4. We will reject the null hypothesis when \(|z|\) is sufficiently small

  2. Determine the rejection region for this test at the 3% level of significance

  3. (5810/SISS students only) Calculate the \(p\)-value for this test

17.2.1 Probability

You flip three fair coins (i.e. the probability that a coin comes up heads is 50%), and count the number of coins that do not match. Let this number be \(X\). For example:

  • If you flipped three heads, then \(X=0\) (because they all match)
  • If you flipped one head then two tails, then \(X=2\) because the first two do not match, the first and third do not match, but the second and third do match.

Hint: Write out a table showing the \(2^3=8\) possible combinations of 3 coin flips you could observe, the probability of each of these occurring, and the value of \(X\) in each of these cases.

  1. What is the support of \(X\)?
  2. Is \(X\) a discrete or continuous random variable? Briefly explain your answer.
  3. What is the probability mass/density function of \(X\)?
  4. What is \(E[X]\), the expectation of \(X\)?
  5. What is \(V[X]\), the variance of \(X\)?

17.2.2 Estimation and inference

Consider the geometric distribution, which has the following properties:

\[ \begin{aligned} \text{parameter: } \theta&\in (0,1)\\ \text{pmf: } p_X(x)&=\begin{cases}(1-\theta)^{x-1}\theta &\text{ if }x = 1, 2, 3,\ldots\\ 0&\text{otherwise} \end{cases}\\ \text{cdf: } F_X(x)&=1-(1-\theta)^x\quad \text{ (for positive integers)}\\ E[X]&=\frac{1}{\theta}\\ V[X]&=\frac{1-\theta}{\theta^2} \end{aligned} \]

You can think of the geometric distribution as follows: You flip a coin that comes up heads with probability \(\theta\) and tails with probability \((1-\theta)\) until you first flip heads. \(X\) is the number of times you flip the coin.

  1. Suppose you flip your coin 10 times before stopping (i.e. \(X=10\)). Using this as your test statistic, what is the \(p\)-value for the following hypothesis test?

\[ H_0: \ \theta=0.5, \quad H_A:\ \theta<0.5 \]

Now suppose that you have a sample of 1,000 draws from this distribution with a sample mean of \(\bar x=5\).

  1. Based on the expression for \(E[X]\), propose an estimator for \(\theta\). Briefly explain why it might make sense to use this estimator.

  2. Using the above information, write down a test statistic that is approximately standard normal (i.e. \(N(0,1)\)) for the following null and alternative hypotheses:

\[ H_0: \ \theta=0.3, \quad H_A:\ \theta\neq0.3 \]

  1. (5810 students only) Approximate the variance of the estimator you proposed in part (2).

17.7.1 Maximum likelihood

The logarithmic distribution is a discrete probability distribution with support \(k\in\{1, 2, 3, 4,\ldots\}\). Its probability mass function is:

\[ \begin{aligned} p(k)&=\begin{cases} \frac{-1}{\log(1-\rho)}\frac{\rho^k}{k}&\text{if }k=1, 2, 3, 4 \ldots\\ 0&\text{otherwise} \end{cases} \end{aligned} \]

where \(\rho\in(0,1)\) is the distribution’s parameter. Note that since \(\rho\in(0,1)\), it follows that \(\frac{-1}{\log(1-\rho)}>0\).

Suppose that you have a sample of data from this distribution \(\{k_i\}_{i=1}^n\), and you need to estimate the parameter \(\rho\).

  1. Write down the likelihood function.

  2. Convert your answer into the log-likelihood function

  3. Determine the first-order condition that implicitly defines the maximum likelihood estimator \(\hat\rho\). Do not try to solve for \(\hat\rho\) analytically, this is one we would solve numerically.

17.8.2 Causal inference

According to Wikipedia:

Maimonides’ rule is named after the 12th-century rabbinic scholar Maimonides, who identified a correlation between class size and students’ achievements. Today this rule is widely used in educational research to evaluate the effect of class size on students’ test scores. Maimonides’ rule states that a class size may rise to an upper limit of 40 students. Once this quota is reached the class is cut in half, so instead of one class with forty-one students there are now two classes: one with twenty students and one with twenty-one students.

This rule is still observed in many Israeli schools today. For example, this figure shows data from 3rd grade classes in Israel. Each dot represents a 3rd grade class, and the line shows the predicted class sizes if Maimonides’ rule was adhered to perfectly (and classes are evenly sized). Note, however, that the rule is not followed perfectly. You can see this because there are come schools with enrollment of fewer than 40 3rd graders, but the class size is not equal to enrollment.
