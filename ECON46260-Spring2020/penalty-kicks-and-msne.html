9.2 MSNE - empirical payoffs

Note that we can solve for MSNE by noting that all expected payoffs must be constant, and that probabilities must add to 1:

\[ \begin{aligned} EU(a)&=\text{a constant}\\ U_ip_i&=c\\ p_i&=U_i^{-1}c\\ \text{probabilities must add to 1:} \\ p_i&=U_i^{-1}c / \sum_a U_i^{-1}c = U_i^{-1}c / \sum_a U_i^{-1}c \end{aligned} \] Only works if payoffs are all non-negative, but we can always add a large constant to \(U\) to fix this, and not change any of the incentives.

9.3 Testing the assumptions implied by aggregation

\[H_0:\ \theta_1=\theta_2,\quad H_A: \theta_1>\theta_2\]

\[ \begin{aligned} p&=\frac{1}{n}\sum_{i=1}^n x_i\\ \sqrt n(\bar x-E[X])&\xrightarrow[]{d}N(0,V[X])\\ \\ \sqrt{n}(\hat p_k-\theta_k)\xrightarrow[]{d}N(0,\theta_k(1-\theta_k))\\ \text{if null is true:}& \ \theta_k=\theta\\ \text{given }H_0: \sqrt{n}(\hat p_1-\theta)-\sqrt{n}(\hat p_2-\theta)\xrightarrow[]{d}N(0,2\theta(1-\theta))\\ \frac{\sqrt{n}(\hat p_1-\hat p_2)}{\sqrt{2\theta(1-\theta)}}&\sim \mathrm{approx} N(0,1) \end{aligned} \]
